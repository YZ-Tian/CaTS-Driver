# Data

- Mutation data can be downloaded from public databases: [TCGA](https://www.cancer.gov/ccg/research/genome-sequencing/tcga) and [IntOGen](https://www.intogen.org/download)

- Macroscopic Genomic Features can be downloaded from [GTEx](https://www.gtexportal.org/home/), [STRING](https://string-db.org/), [gnomAD](https://gnomad.broadinstitute.org/), [MutSigCV](https://www.genepattern.org/modules/docs/MutSigCV/1/)
