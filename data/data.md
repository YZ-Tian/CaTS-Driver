# Data

- Mutation data can be downloaded from public databases: [TCGA]([The Cancer Genome Atlas Program (TCGA) - NCI](https://www.cancer.gov/ccg/research/genome-sequencing/tcga)) and [IntOGen]([IntOGen - Cancer Mutations Browser](https://www.intogen.org/download))

- Macroscopic Genomic Features can be downloaded from [GTEx]([GTEx Portal](https://www.gtexportal.org/home/)), [STRING]([STRING: functional protein association networks](https://string-db.org/)), [gnomAD](https://gnomad.broadinstitute.org/), [MutSigCV]([GenePattern - MutSigCV (v1)](https://www.genepattern.org/modules/docs/MutSigCV/1/)),
